# ShoePedi User Documentation

## 1) Getting started
1. Open the website home page.
2. Browse featured products, categories, brands, and tags.
3. Use search to find products quickly.

## 2) Create an account / sign in
- Go to **Sign Up** to create an account.
- Verify your email when prompted.
- Use **Sign In** to access account features.
- If needed, use **Forgot Password** to reset your password.

## 3) Browse products
You can discover products by:
- **Categories** (`/categories`)
- **Brands** (`/brands`)
- **Tags** (`/tags`)
- **Search** (`/search`)
- **Home recommendations** (best sellers, new arrivals, today’s deals)

## 4) Product page actions
On each product page, you can:
- Select size/color (if applicable)
- Add product to cart
- Read and write reviews
- Compare with other products
- Add to wishlist

## 5) Cart and checkout
1. Open cart and verify quantities/variants.
2. Proceed to checkout.
3. Enter shipping and contact details.
4. Complete payment (Paystack-enabled flow).
5. Receive order confirmation.

## 6) Track orders
- Use the tracking page at `/track`.
- Enter your tracking number to view order progress.

## 7) Account area
Under **My Account** (`/account`) users can manage:
- Profile details
- Saved addresses
- Orders
- Reviews
- Comments
- Wishlist
- Support requests
- Coins (if enabled)

## 8) Additional features
- **Affiliate program:** register and monitor dashboard/payouts.
- **Blogs and pages:** read store blog posts and site information pages.
- **Newsletter:** subscribe/unsubscribe from updates.

## 9) Troubleshooting
- If you cannot sign in, reset your password and verify email status.
- If checkout fails, confirm payment details and try again.
- If you need help, use the support form in your account or the public support page.
